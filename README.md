 - Selling Management
 - Per Shop Analytics

 - NodeJs(Express)
 - ReactJs
 - Formik

 - Redux state management 
 - Tailwindcss
 - Firebase
 - SASS
Deployment Platform : 
 - Kubernetes 
 - GCP
